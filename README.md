# VKAdmin — AI-администратор для групп ВКонтакте

Полноценный бот-администратор с ИИ для управления сообществами VK. Автоматическая модерация, генерация контента, геймификация, аналитика и кросс-постинг в Telegram.

## Возможности

### Для участников
- **Геймификация** — XP за сообщения, лайки, репосты; уровни и достижения
- **Личный кабинет** — профиль с прогресс-баром, статистикой и бейджами
- **Лидерборд** — рейтинг топ-50 с фильтрами (XP, уровень, репутация)
- **Магазин** — VIP-статус и внутренняя валюта
- **Предложка** — пользователи предлагают посты, админ одобряет одним кликом

### Для администраторов
- **AI-контент** — генерация постов, статей из URL, патч-нотов из GitHub
- **Автопостинг** — парсинг RSS/VK/сайтов + ИИ-переписывание
- **Умная модерация** — AI-анализ токсичности комментариев, авто-баны
- **Контент-план** — визуальный календарь на 7 дней, планирование
- **Аналитика** — лайки, просмотры, репосты, топ участников
- **Рассылка** — отправка сообщений всем участникам
- **Виджет** — таблица лидеров прямо на странице группы

### Интеграции
- **VK Mini App** — полноценное приложение внутри VK с навигацией
- **Telegram** — автоматический кросс-постинг в канал
- **GitHub** — патч-ноты из коммитов (приватные репо)
- **OpenRouter / свой AI** — любой LLM-провайдер

## Архитектура

```
FastAPI (uvicorn) — веб-сервер
├── VK Callback API — приём событий от VK
├── Mini App — пользовательский интерфейс внутри VK
├── Dashboard — веб-панель администратора
├── Public API — REST API с авторизацией
└── APScheduler — фоновые задачи
    ├── Автопостинг (каждый час)
    ├── Публикация запланированных постов (каждые 5 мин)
    ├── Контент-задачи по cron
    ├── Аналитика (каждые 6 часов)
    └── Обновление виджета (каждый час)
```

**Multi-tenant** — один инстанс обслуживает множество групп. Каждая группа изолирована через `group_id`.

## Быстрый старт

### 1. Установка

```bash
git clone https://github.com/Leonid1095/PLGames-VKAdmin.git
cd PLGames-VKAdmin
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Настройка

```bash
cp .env.example .env
# Заполните .env — минимально нужны:
# VK_APP_ID, VK_APP_SECRET, OPENROUTER_API_KEY, ENCRYPTION_KEY, JWT_SECRET
```

Генерация ключа шифрования:
```bash
python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
```

### 3. Запуск

```bash
python main.py
# Сервер на http://localhost:8000
```

### 4. Подключение группы

1. Откройте `https://yourdomain.com/dashboard`
2. Введите пароль (= `JWT_SECRET` из `.env`)
3. Укажите ID группы и нажмите "Подключить"
4. Авторизуйте приложение в VK

## Команды бота

### Админ-команды (/)
| Команда | Описание |
|---------|----------|
| `/пост [тема]` | Сгенерировать и опубликовать AI-пост |
| `/статья <url>` | Написать статью из URL |
| `/патчнот <github>` | Патч-ноты из коммитов |
| `/запланировать ЧЧ:ММ текст` | Запланировать пост |
| `/контентплан` | Показать план на сегодня |
| `/предложка` | Предложения пользователей |
| `/принять <id>` | Опубликовать предложение |
| `/стата` | Статистика группы |
| `/аналитика` | Аналитика постов |
| `/топ [xp\|rep\|messages]` | Рейтинг участников |
| `/бан <vk_id>` | Забанить |
| `/рассылка <текст>` | Рассылка всем участникам |
| `/виджет` | Обновить виджет |

### Пользовательские команды (!)
| Команда | Описание |
|---------|----------|
| `!профиль` | Личный кабинет |
| `!купить` | Магазин VIP и коинов |
| `!предложить <текст>` | Предложить пост |
| `!гороскоп` | Шуточный гороскоп от ИИ |
| `!кто я` | ИИ-анализ личности |

## Переменные окружения

| Переменная | Обязательно | Описание |
|-----------|------------|----------|
| `VK_APP_ID` | Да | ID приложения VK |
| `VK_APP_SECRET` | Да | Секрет приложения VK |
| `OPENROUTER_API_KEY` | Да | API-ключ AI-провайдера |
| `ENCRYPTION_KEY` | Да | Fernet-ключ для шифрования токенов |
| `JWT_SECRET` | Да | Пароль для dashboard |
| `BASE_URL` | Да | Публичный URL сервера |
| `VK_MINIAPP_ID` | Нет | ID VK Mini App |
| `VK_MINIAPP_SECRET` | Нет | Секрет Mini App |
| `GITHUB_TOKEN` | Нет | GitHub PAT для приватных репо |
| `TELEGRAM_BOT_TOKEN` | Нет | Токен Telegram-бота |
| `API_KEY` | Нет | Ключ для публичного API |
| `DATABASE_URL` | Нет | URL БД (по умолч. SQLite) |

## Деплой (systemd)

```bash
sudo cp vkadmin.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable vkadmin
sudo systemctl start vkadmin
```

## Технологии

- **Python 3.12** + FastAPI + uvicorn
- **SQLAlchemy 2.0** (async) + SQLite / PostgreSQL
- **vkbottle** — VK API
- **OpenAI SDK** — совместимый с OpenRouter, собственными AI
- **APScheduler** — фоновые задачи
- **httpx** — async HTTP-клиент
